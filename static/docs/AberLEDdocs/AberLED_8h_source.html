49 /// The class for the AberLED shield. One object of this class, called
50 /// **AberLED**, is automatically created (similar to how Serial works).
51 /// Yes, all this time you've been writing C++, not C. You should put
52 /// `AberLED.` before every function, so `AberLED.begin()` instead
53 /// of just `begin()`.
54 ///
55 ///
56 /// Calling begin() will initialise the LED matrix , setting all the pins to
57 /// the appropriate mode and setting up the interrupt handler.
58 /// Thereafter, every nth of a second the Arduino will automatically
59 /// send the front buffer to the matrix.
60 /// Calling getBuffer() will give a pointer to the back buffer, which
61 /// can be be written to. Alternatively, pixels can be written by calling
62 /// set() with x,y coordinates and a colour (GREEN, RED, YELLOW or BLACK).
63 /// Once writing is complete, calling swap() will
64 /// exchange the front and back buffers, so that the Arduino will
65 /// now write the new front buffer to the matrix every interrupt tick.
66 ///
67 /// The format of the buffers is 8 16-bit integers: unsigned shorts.
68 /// Each integer represents a single row. Each pixel - individual LED -
69 /// is represented by 2 adjacent bits with the most significant bit
70 /// being red and the least significant bit being green.
71
